## 텍스트 편집기

- 대부분의 Linux 시스템은 서버로 사용되어 GUI가 없다. 또한 거의 구성 파일로 이루어져 있어 변경하려면, 구성 파일을 편집해야 한다.
- 다양한 옵션이 있지만, 그 중에서 빠른 수정 작업에는 `nano`가 편리하고 `vim`은 다양한 기능을 제공하여 더 복잡한 편집 작업을 수행할 수 있다.

### nano

```shell
nano 17day.txt # 새 파일 생성, Ctrl + O : 저장, Ctrl + X : 종료
```

### vim

```shell
vim 17day.txt # 새 파일 생성
```

- command, normal, visual, insert 모드가 존재한다. `i`, `a`를 눌러 `normal` 모드에서 `insert`모드로 전환할 수 있다. `ESC`를 누르고 `:wq`를 입력하면
  저장된다.
- `:%s/day/17day`를 입력하면 모든 `day`가 `17day`로 변경된다.
- `normal` 모드에서 `yy`로 복사하고, (같은 줄: `p` / 새 줄: `P`)로 붙여 넣을 수 있다.
- 선택한 줄 삭제: `dd`
- `/특정 단어`로 단어를 찾고, `n`을 누르면 다음 일치 항목으로 이동한다.
- vim에서 방향키 대신 `H, J, K, L`을 사용할 수 있다.
